Plugins

SilkSpawners

 Allows you to mine spawners with a silk touch pick so that they can be replaced elsewhere. This plugin is always enabled and requires no commands.

CoreProtect

 This plugin is for logging all player activity. We are capable of seeing all block/chest interaction so that we can prevent and roll back incidents of greifing and theft.
